# HKT Apron Management Dashboard

This dashboard provides real-time analytics and movement tracking for HKT Apron operations.

## Deployment to GitHub Pages

1. Create a new repository on GitHub (e.g., `hkt-apron`).
2. Upload the contents of this folder (`index.html`, `style.css`, `app.js`).
3. Go to **Settings** > **Pages**.
4. Select `main` branch and `/root` as source.
5. Your dashboard will be live at `https://<your-username>.github.io/hkt-apron/`.

## Data Source
The dashboard synchronizes with Google Sheets automatically using the CSV export API.
- **Master Data**: For movements and monthly trends.
- **Movement Logs**: For bay change analysis.
